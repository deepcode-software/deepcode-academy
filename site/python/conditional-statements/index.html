TARMOQLANISH

üìå Shu vaqtgacha yozgan dasturlarimizga e'tibor bersangiz, dasturimiz yuqoridan pastga qarab qatorma-qator bajarilib keldi. Bu chiziqli dastur deyiladi. Ammo ba'zida shartga qarab kodning bir qismidan boshqa qismiga o'tish zarur bo'ladi. Bunday holat "tarmoqlanish" deb ataladi.

üìå Python dasturlash tilida shart operatorlari (conditional statements) dasturda turli vaziyatlarga qarab turli amallarni bajarishga imkon beradi.

‚úÖ IF

üìå Python dasturlash tilida if operatori shartni tekshirish uchun ishlatiladi. Ya'ni, agar biror shart True bo‚Äòlsa, unga tegishli kodlar bajariladi. Agar shart False bo‚Äòlsa, kod bajarilmaydi.

# Agar balans 1000 dan katta bo‚Äòlsa, foydalanuvchi pul yechishi mumkin
if account_balance > 1000:
    print("You can withdraw money")  # Foydalanuvchiga ruxsat beriladi

# Foydalanuvchining savatidagi umumiy mahsulot narxi
total_price = 250  

# Agar narx 200 dan katta bo‚Äòlsa, bepul yetkazib berish taklif qilinadi
if total_price > 200:
    print("You qualify for free shipping")  # Bepul yetkazib berish haqida xabar

# Foydalanuvchi kiritgan parol
password = "mysecurepass"  

# Agar parol uzunligi 8 belgidan ko‚Äòp bo‚Äòlsa, kuchli parol deb baholanadi
if len(password) > 8:
    print("Your password is strong")  # Kuchli parol haqida xabar

‚úÖ ELSE

üìå Python dasturlash tilida else operatori if dan keyin yoziladi. Agar if dagi False bo;lsa, else ichidagi kodlar bajariladi.

‚úÖ ELIF

elif ‚Äî bu else if degan so‚Äòzning qisqargan ko‚Äòrinishi. U bir nechta shartlarni ketma-ket tekshirish uchun ishlatiladi. Agar if sharti False bo'lsa, elif sharti tekshiriladi.

‚úÖ NESTED CONDITIONS

üìå Shart operatorlarini bir-birining ichiga joylashtirish orqali murakkabroq mantiqiy holatlar yaratish mumkin.

üéØ Foydalanuvchi login kiritadi, agar mavjud bo‚Äòlsa, yoshi tekshiriladi. Faqat voyaga yetganlarga ruxsat.

# Foydalanuvchi nomi (sistemaga kirishga urinayotgan)
username = "umid"

# Foydalanuvchilar ro'yxati (username: age)
users = {
    "umid": 21,
    "ali": 17
}

# Avvalo foydalanuvchi ro'yxatda bormi ‚Äì tekshiramiz
if username in users:
    age = users[username]  # Username bo‚Äòyicha yoshni olamiz

    # Agar foydalanuvchi 18 yoki undan katta bo‚Äòlsa ‚Äì ruxsat beriladi
    if age >= 18:
        print(f"{username.title()} is allowed to access the system.")  # Ruxsat berildi

    # Agar foydalanuvchi voyaga yetmagan bo‚Äòlsa ‚Äì ruxsat berilmaydi
    else:
        print(f"{username.title()} is underage. Access denied.")  # Ruxsat yo‚Äòq
else:
    print("User not found in the system.")  # Bunday foydalanuvchi yo‚Äòq

üéØ Foydalanuvchi balansiga qarab faqat ruxsat etilgan tranzaksiyani bajarish mumkin.

# User's current balance

# Transaction type: "withdraw" (yechish) yoki "deposit" (kiritish)

    if balance >= amount:

    print(f"{amount} so'm has been added to your balance.")  # Pul kiritildi

# Aks holda, tranzaksiya turi noto‚Äòg‚Äòri
else:
    print("Invalid transaction type.")  # Noma'lum amal

üéØ Agar foydalanuvchi o‚Äòqituvchi bo‚Äòlsa, yangi kurs yarata oladi. Talaba bo‚Äòlsa, faqat yozilishi mumkin.

‚úÖ MULTIPLE LEVELS OF NESTED CONDITIONAL STATEMENTS

üéØ Ma'lumotlar bazasida foydalanuvchi ro‚Äòyxatdan o‚Äòtganmi, email tasdiqlanganmi va yoshi to‚Äòliqmi ‚Äî shu tartibda tekshiriladi:

    "username": "umid",
    "registered": True,         # Foydalanuvchi ro'yxatdan o'tganmi?
    "email_verified": True,     # Email tasdiqlanganmi?
    "age": 19                   # Foydalanuvchi yoshi
}

# 1. Avval foydalanuvchi ro‚Äòyxatdan o‚Äòtganligini tekshiramiz

    # 2. Email tasdiqlanganligini tekshiramiz
    if user["email_verified"]:

        # 3. Yoshi 18 yoki undan katta bo‚Äòlsa ‚Äì to‚Äòliq ruxsat beriladi
        if user["age"] >= 18:
            print("Full access granted to the system.")  # To‚Äòliq ruxsat

        else:
            print("Access denied. You must be at least 18 years old.")  # Yoshi yetmaydi

    else:
        print("Email not verified. Please verify your email first.")  # Email tasdiqlanmagan

    print("You are not registered. Please sign up first.")  # Ro‚Äòyxatdan o‚Äòtmagan

üéØ Avtomobil ijarasi faqat quyidagi shartlar bajarilganda mumkin:

  • Foydalanuvchi ro‚Äòyxatdan o‚Äòtgan
  • Haydovchilik guvohnomasi mavjud

    "age": 25                 # Foydalanuvchining yoshi

    # 2. Haydovchilik guvohnomasi mavjud bo‚Äòlishi kerak

        # 3. Yoshi 21 dan katta yoki teng bo‚Äòlishi kerak
        if user["age"] >= 21:
            print("Car rental approved.")  # Ijaraga ruxsat berildi
        else:
            print("You must be at least 21 years old to rent a car.")  # Yoshi yetmaydi

    else:
        print("You must have a valid driver‚Äôs license.")  # Guvohnoma yo‚Äòq

else:
    print("Please register before renting a car.")  # Ro‚Äòyxatdan o‚Äòtmagan

‚úÖ AMALIYOT

‚úÖ 1-topshiriq:

Foydalanuvchidan yoshini so‚Äòrang. Agar 18 dan kichik bo‚Äòlsa "Voyaga yetmagan", 18 yoki undan katta bo‚Äòlsa "Voyaga yetgan" deb chiqaring.

‚úÖ 2-topshiriq:

Quyidagi o‚Äòzgaruvchi berilgan:

Agar score 90 va undan yuqori bo‚Äòlsa "A", 80-89 oralig‚Äòida bo‚Äòlsa "B", 70-79 oralig‚Äòida bo‚Äòlsa "C", boshqacha bo‚Äòlsa "D" baho chiqaring.

‚úÖ 3-topshiriq:

Quyidagi o‚Äòzgaruvchi berilgan:

Agar is_raining True bo‚Äòlsa "Soyabon oling", aks holda "Yomg‚Äòir yo‚Äòq" deb chiqaring.

‚úÖ 4-topshiriq:

Quyidagi ro‚Äòyxat berilgan: 

fruits = ["olma", "banan", "gilos"]

Agar ro‚Äòyxatda "banan" bo‚Äòlsa "Banan bor", bo‚Äòlmasa "Banan yo‚Äòq" deb chiqaring.

‚úÖ 5-topshiriq:

Ro‚Äòyxatda kamida 5 ta element bo‚Äòlsa "Ro‚Äòyxat to‚Äòla", aks holda "Ro‚Äòyxat qisqa" deb chiqaring:

‚úÖ 6-topshiriq:

Tuple berilgan:

Agar location bo‚Äòsh bo‚Äòlmasa "Joylashuv aniqlangan", bo‚Äòsh bo‚Äòlsa "Joylashuv yo‚Äòq" deb chiqaring.

‚úÖ 7-topshiriq:

Set berilgan:

Agar 2-ID setda mavjud bo‚Äòlsa "2-ID mavjud", bo‚Äòlmasa "2-ID yo‚Äòq" deb chiqaring.

‚úÖ 8-topshiriq:

Quyidagi dictionary berilgan: 

profile = {"name": "Alim", "age": 15}

Agar age 18 yoki undan katta bo‚Äòlsa "Voyaga yetgan", aks holda "Voyaga yetmagan" deb chiqaring.

‚úÖ 9-topshiriq:

Quyidagi o‚Äòzgaruvchilar berilgan:

is_admin = False

Agar foydalanuvchi tizimga kirgan va admin bo‚Äòlsa "Admin panelga xush kelibsiz", faqat tizimga kirgan bo‚Äòlsa "Foydalanuvchi paneli", har ikkisi ham False bo‚Äòlsa "Kirish talab qilinadi".

‚úÖ 10-topshiriq:

Quyidagi kodni if...elif...else bilan yozing:

- 0 dan past "Muzlab qolgan"
- 0-15 oralig‚Äòida "Salqin"
- 16-30 oralig‚Äòida "Me'yoriy"
- 30 dan yuqori "Issiq"

‚úÖ 11-topshiriq:

Ro‚Äòyxatda "python" va "sql" bo‚Äòlsa "Ikkalasi ham bor", faqat bittasi bo‚Äòlsa "Bittasi bor", ikkalasi ham bo‚Äòlmasa "Ikkalasi ham yo‚Äòq" deb chiqaring:

‚úÖ 12-topshiriq:

Foydalanuvchi parol kiritadi. Agar parol uzunligi 10 dan katta bo‚Äòlsa "Kuchli parol", 6-10 oralig‚Äòida bo‚Äòlsa "O‚Äòrtacha parol", 6 dan kam bo‚Äòlsa "Juda qisqa".

‚úÖ 13-topshiriq:

Quyidagi dictionary berilgan: 

user = {"username": "nargiza", "active": True, "role": "student"}

Agar user active va roli "teacher" bo‚Äòlsa "Yangi kurs yarata oladi", active va roli "student" bo‚Äòlsa "Faqat kursga yozila oladi", aks holda "Ruxsat yo‚Äòq".

‚úÖ 14-topshiriq:

Quyidagi o‚Äòzgaruvchilar berilgan:

is_connected = True

Agar is_connected False bo‚Äòlsa "Internet yo‚Äòq", else if speed < 1 bo‚Äòlsa "Eng past sifat", speed < 5 bo‚Äòlsa "O‚Äòrtacha sifat", else "Yuqori sifat".

‚úÖ 15-topshiriq:

Quyidagi dictionarylar orqali avtomobil ijarasi uchun quyidagilarni tekshiring:
- "registered": True
- "has_license": True
- "age": 19
Agar hammasi to‚Äòg‚Äòri bo‚Äòlsa "Ijaraga ruxsat", aks holda qaysi shart bajarilmaganini chiqaring.
