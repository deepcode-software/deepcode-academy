üìå Tuple ‚Äî bu Pythonda bir nechta ma‚Äôlumotni bitta o‚Äòzgaruvchida saqlash uchun ishlatiladigan o‚Äòzgarmas tuzilma. U dumaloq qavs ichida yoziladi va elementlar vergul bilan ajratiladi. Tuple yaratilgach, uning ichidagi qiymatlarni o‚Äòzgartirib, o‚Äòchirib yoki yangisini qo‚Äòshib bo‚Äòlmaydi. Undagi ma‚Äôlumotlar tartib bilan saqlanadi va indeks orqali chaqiriladi. Tuple listga o‚Äòxshaydi, lekin o‚Äòzgarmasligi bilan farq qiladi. U dasturda tezroq ishlaydi va kamroq xotira egallaydi. O‚Äòzgarmas ma‚Äôlumotlarni xavfsiz saqlash uchun tuple juda qulay.

‚úÖ TUPLE XUSUSIYATLARI

üìå O‚Äòzgarmaydi: Tuple yaratilgandan keyin uning ichidagi ma‚Äôlumotlarni o‚Äòzgartirib bo‚Äòlmaydi. Masalan, yangi element qo‚Äòshib yoki mavjudini o‚Äòchirib bo‚Äòlmaydi. Bu degani ‚Äî tuple ichidagi ma‚Äôlumotlar doim bir xil bo‚Äòlib qoladi.

üìå Tartibli: Tuple ichida ma‚Äôlumotlar tartib bilan saqlanadi. Har bir element o‚Äòz o‚Äòrniga ega. Siz bu o‚Äòrinni raqam bilan chaqirishingiz mumkin (masalan: birinchi element, ikkinchi element va hokazo).

üìå Har xil turdagi ma‚Äôlumot bo‚Äòlishi mumkin: Tuple ichida sonlar, matnlar, True/False qiymatlar yoki boshqa turdagi ma‚Äôlumotlar aralash bo‚Äòlishi mumkin. Bularni bir joyda saqlash imkonini beradi.

üìå Tez ishlaydi: Tuple ‚Äî kompyuter uchun engil va tez tuzilma. Dastur ishlaganda tuplelar ro‚Äòyxatlarga qaraganda tezroq ishlaydi va kam xotira ishlatadi.

üìå O‚Äòzgarmaydigan ma‚Äôlumotlar uchun qulay: Agar sizda haftaning kunlari, oylar nomi, davlatlar ro‚Äòyxati kabi o‚Äòzgarmas ma‚Äôlumotlar bo‚Äòlsa, tuple juda foydali bo‚Äòladi.

‚úÖ TUPLE YARATISH

‚ùáÔ∏è ODDIY TUPLE YARATISH

# Uchta meva nomini o‚Äòz ichiga olgan tuple yaratamiz
fruits = ("apple", "banana", "cherry")

# fruits tuple ichidagi barcha elementlarni ekranga chiqaramiz
print(fruits)

‚ùáÔ∏è BITTA E'LEMENTLI TUPLE YARATISH

# Faqat bitta elementdan iborat tuple yaratamiz
# E‚Äôtibor bering, oxirida vergul qo‚Äòyilishi shart
single_fruit = ("apple",)

# Bitta elementli tuple ni ekranga chiqaramiz
print(single_fruit)



# ‚ùå Bu tuple emas (string bo‚Äòlib qoladi)

not_a_tuple = ("apple")

# O‚Äòzgaruvchi turi (type) ni tekshiramiz

‚úÖ TUPLE E'LEMENTLARIGA MUROJAT QILISH

üìå Tuple ichidagi har bir element tartib bilan joylashgan va ularga indekslar orqali murojaat qilish mumkin. Hisoblash 0 dan boshlanadi, ya‚Äôni birinchi elementning indeksi 0 bo‚Äòladi. Bu xuddi listdagi kabi ishlaydi.

# Bu yerda 5 ta elementdan iborat tuple yaratilmoqda
my_tuple = (10, 20, 30, 40, 50)

# [0] ‚Äî bu birinchi elementga murojaat, indeks 0 dan boshlanadi
print(my_tuple[0])  # 10

# [2] ‚Äî bu uchinchi elementga murojaat (0 - 1 - 2), qiymati 30
print(my_tuple[2])  # 30

# [-1] ‚Äî bu oxirgi elementga murojaat, -1 doimo oxirgi elementni bildiradi
print(my_tuple[-1])  # 50

‚úÖ TUPLE USTIDA AMALLAR

üìå Tuple ‚Äî bu o‚Äòzgarmas ma‚Äôlumot turi. Ya‚Äôni, yaratilgach, uning ichidagi qiymatlarni o‚Äòzgartirib, o‚Äòchirib yoki yangisini qo‚Äòshib bo‚Äòlmaydi.

  • Lekin, siz tuple bilan ikkita amalni bemalol bajara olasiz:
    • Tuplelarni birlashtirish
    • Tupleni takrorlash

‚ú≥Ô∏è + OPERATORI

üìå Bu usulda ikkita tuple birga qo‚Äòshiladi va yangi tuple hosil bo‚Äòladi. Asl tuplelar o‚Äòzgarmaydi.

# Birinchi tuple
tuple1 = (1, 2)

# Ikkinchi tuple
tuple2 = (3, 4)

# Ikkalasini birlashtiramiz va yangi tuple hosil qilamiz
new_tuple = tuple1 + tuple2

# Natijani chiqaramiz
print(new_tuple)  # (1, 2, 3, 4)

‚ùó Yuqorida tuple1 va tuple2 o‚Äòzgarmagan holda qoladi, faqat + orqali ularning qiymatlari birga qo‚Äòshilib, new_tuple degan yangi tuple yaratiladi.

‚ú≥Ô∏è TAKRORLASH

üìå Agar bir tuple ni o‚Äòzini o‚Äòziga birlashtirmoqchi bo‚Äòlsangiz, uni yana o‚Äòzi bilan + orqali birlashtirasiz.

# Asl tuple
tuple1 = (1, 2, 3)

# O‚Äòz-o‚Äòziga birlashtirish (ikki marta yozilgan bo‚Äòladi)
result = tuple1 + tuple1

# Natijani chiqaramiz
print(result)  # (1, 2, 3, 1, 2, 3)

‚ùáÔ∏è TUPLENI KO'PAYTIRISH (TAKRORLASH)

üìå Tuple o‚Äòzgarmas bo‚Äòlsa ham, uni bir nechta marta takrorlab yangi tuple yaratish mumkin. Buning uchun * operatoridan foydalaniladi.

# Bitta elementdan iborat tuple yaratilmoqda
tuple1 = ("hello",)

# Bu tuple 3 marta takrorlanmoqda
new_tuple = tuple1 * 3

# Natijada yangi tuple hosil bo‚Äòladi
print(new_tuple)  # ('hello', 'hello', 'hello')

‚ùáÔ∏è TUPLE UZUNLIGINI ANIQLASH

üìå Tuple ichida nechta element borligini bilish uchun len() funksiyasidan foydalaniladi. Bu funksiya tuple uzunligini, ya‚Äôni elementlar sonini qaytaradi.

# 5 ta elementdan iborat tuple yaratilmoqda
my_tuple = (1, 2, 3, 4, 5)

# len() funksiyasi tuple ichidagi elementlar sonini hisoblaydi
print(len(my_tuple))  # 5

‚ùáÔ∏è in OPERATORI

üìå in operatori yordamida biror qiymat tuple ichida bor yoki yo‚Äòqligini tekshirish mumkin. Natija True yoki False ko‚Äòrinishida chiqadi.

# Mevalar saqlangan tuple yaratilmoqda
my_tuple = ("apple", "banana", "cherry")

# "banana" tuple ichida bormi? degan savolga javob tekshirilmoqda
print("banana" in my_tuple)  # True

‚ùáÔ∏è TUPLENI QIYMATLARGA AJRATISH(Unpacking)

üìå Tuple ichidagi har bir qiymatni alohida o‚Äòzgaruvchiga ajratib olish mumkin. Bunga Pythonda unpacking deyiladi.

# 3 ta meva nomidan iborat tuple yaratilmoqda
my_tuple = ("apple", "banana", "cherry")

# Tuple ichidagi har bir qiymat alohida o‚Äòzgaruvchiga ajratilmoqda
(fruit1, fruit2, fruit3) = my_tuple

# Har bir o‚Äòzgaruvchidagi qiymat ekranga chiqariladi
print(fruit1)  # 'apple'
print(fruit2)  # 'banana'
print(fruit3)  # 'cherry'

‚ùáÔ∏è NESTED TUPLE

üìå Tuple ichida yana boshqa tuple ham saqlanishi mumkin. Bunga ichma-ich tuple yoki nested tuple deyiladi.

# Bu yerda talaba haqida ma'lumotlar saqlanmoqda:
# ism, yosh, va baholar (baholar alohida tuple ichida)
student = ("Ali", 20, (85, 90, 95))

# Indeks 0 ‚Äî ismni beradi
print(student[0])  # Ali

# Indeks 1 ‚Äî yoshni beradi
print(student[1])  # 20

# Indeks 2 ‚Äî baholar joylashgan ichki tuple
print(student[2])  # (85, 90, 95)

# Ichki tuple ichidagi ikkinchi bahoni olish
print(student[2][1])  # 90

‚ùáÔ∏è TUPLE BILAN ISHLASHDA FOYDALI METODLAR

‚ú≥Ô∏è .count()

üìå Bu funksiya siz bergan qiymatni tuple ichida qidiradi va nechta marta borligini sanaydi. Agar qiymat topilmasa, 0 qaytaradi.

# Bir nechta takrorlanuvchi sonlardan iborat tuple
my_tuple = (1, 2, 2, 3, 2)

# 2 soni tuple ichida necha marta borligini hisoblaydi
print(my_tuple.count(2))  # 3

‚ú≥Ô∏è .index()

üìå Bu metod tuple ichida berilgan qiymat qayerda turganini aniqlaydi, ya‚Äôni indeks raqamini qaytaradi.

# Elementlar ketma-ketligi
my_tuple = (1, 2, 3, 2, 4)

# 2 soni birinchi bo‚Äòlib qayerda uchraganini topadi (indeks 1)
print(my_tuple.index(2))  # 1
