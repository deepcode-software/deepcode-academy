üß© 11-DARS FUNCTIONS

üìå Python dasturlash tilida funksiya bu ‚Äî kodni bir joyda yozib, ko‚Äòp joylarda chaqirish, kodni tartibli va qisqa qilish uchun qulay vositadir. Funksiya yordamida kodni modullashtirish va takrorlanadigan qismni soddalashtirish mumkin.

‚úÖ FUNKSIYA YARATISH (DEF)

def greet():
    # Oddiy funksiya: salomlashish
    print("Hello, world!")

‚úÖ CALLING A FUNCTION (FUNKSIYANI CHAQIRISH)

üìå Pythonda yozilgan funksiyani ishlatish uchun uni chaqirish kerak bo‚Äòladi. Buning uchun faqat funksiyaning nomi va qavslar () yoziladi.

# Bu yerda biz greet() degan nomli funksiya yaratdik
def greet():
    # Funksiya ichida "Hello, world!" degan matnni chiqaradigan buyruq bor
    print("Hello, world!")

# Bu yerda esa yuqorida yaratilgan greet() funksiyasini chaqiryapmiz
greet()

‚úÖ PARAMETERS AND ARGUMENTS (PARAMETRLAR VA ARGUMENTLAR)

üìå Parametr ‚Äì bu funksiya yaratilyotganda yoziladigan o‚Äòzgaruvchilar bo‚Äòlib, ular funksiyaga ma‚Äôlumot qabul qilish uchun ishlatiladi.

# Funksiya yaratilyapti, u 2 ta parametr oladi: a va b
def add(a, b):
    # a va b sonlar qo‚Äòshilib, natija qaytariladi
    return a + b

# Funksiyani chaqiryapmiz, 2 va 3 argument sifatida uzatilyapti
result = add(2, 3)

# Natijani chiqaramiz
print(result)  # Natija: 5

üéØ Tasavvur qilaylik, siz foydalanuvchi ma‚Äôlumotlarini qabul qilib, uni ro‚Äòyxatga qo‚Äòshishingiz kerak.

# Bo‚Äòsh foydalanuvchilar ro‚Äòyxati
users = []

# Funksiya foydalanuvchi ma‚Äôlumotlarini qabul qiladi
def add_user(name, age):

    }

add_user("Laylo", 22)

# Natijada users ro‚Äòyxatida 2 ta foydalanuvchi bo‚Äòladi
print(users)

‚úÖ DEFAULT PARAMETERS (STANDART PARAMETRLAR)

üìå Standart parametr ‚Äî bu funksiyada oldindan belgilangan qiymatga ega bo‚Äòlgan parametr. Agar funksiyani chaqirishda bu parametr uchun argument berilmasa, u default qiymatni ishlatadi.

# Funksiya yaratildi, exponent parametri uchun default qiymat 2
def power(base, exponent=2):
    # base son exponent darajaga ko‚Äòtariladi
    return base ** exponent

# Faqat bitta argument berdik ‚Äì base = 3, exponent 2 deb olinadi (default)
print(power(3))      # Natija: 9 (3^2)

# Ikkala argumentni ham berdik ‚Äì base = 3, exponent = 3
print(power(3, 3))   # Natija: 27 (3^3)

üéØ Sizda parol yaratish funksiyasi bor. Unga uzunlik (length) parametri beriladi, lekin foydalanuvchi kiritmasa, standart qiymat sifatida 8 belgilanadi.

import random
import string

# Parol yaratish funksiyasi: uzunlik uchun standart qiymat 8
def generate_password(length=8):
    belgilar = string.ascii_letters + string.digits
    parol = ''.join(random.choice(belgilar) for _ in range(length))
    return parol

# Default uzunlik bilan chaqirish
print(generate_password())      # Masalan: a7B3kL2q

# Uzunlikni o‚Äòzimiz beramiz
print(generate_password(12))    # Masalan: jD93kLm8Tz2Q

‚úÖ RETURN (QIYMAT QAYTARISH)

üìå return ‚Äî bu funksiya natijasini (qiymatini) tashqariga qaytarish uchun ishlatiladi. Bu qiymatni keyin saqlash, chiqarish yoki boshqa hisob-kitoblarda ishlatish mumkin.

# Funksiya: x va y ni ko‚Äòpaytiradi
def multiply(x, y):
    return x * y  # Natija return orqali qaytariladi

# multiply(4, 5) = 20, bu qiymat product o‚Äòzgaruvchisiga saqlanadi
product = multiply(4, 5)

# Natijani ekranga chiqaramiz
print(product)  # 20

‚úÖ VOID FUNCTION (HECH NIMA QAYTARMAYDIGAN FUNKSIYALAR)

# Funksiya: foydalanuvchini kutib oladi

‚úÖ VALUE-RETURNING FUNCTION (QIYMAT QAYTARADIGAN FUNKSIYALAR)

üìå Qiymat qaytaradigan funksiya ‚Äî bu funksiya return operatori yordamida natijani (qiymatni) qaytaradi.

# Funksiya: ikki sonning kattasini aniqlaydi
def get_maximum(a, b):
    if a > b:
        return a  # Agar a katta bo‚Äòlsa, a ni qaytar
    else:
        return b  # Aks holda b ni qaytar

# Funksiyani chaqiramiz, 7 va 10 ni argument sifatida uzatamiz

# Natijani chiqaramiz
print(max_number)  # 10

üéØ Siz onlayn do‚Äòkon qilayapsiz va sizga mahsulotlar ichidan eng qimmatini topuvchi funksiya kerak.

‚úÖ FUNKSIYAGA RO‚ÄòYXAT (LIST) UZATISH

üìå Pythonda funksiya yaratganda parametr sifatida ro‚Äòyxat (list) berishimiz mumkin. Bu funksiyaga bir nechta qiymatni birdan uzatish imkonini beradi.

‚úÖ *ARGS

üìå *args ‚Äì bu funksiya istalgancha sonli argument qabul qilishi uchun ishlatiladi. U * belgisi bilan yoziladi, va Python uni tuple sifatida saqlaydi.

# Funksiya: istalgancha son qabul qilib, ularning yig‚Äòindisini qaytaradi
def total_sum(*args):
    # args bu tuple ko‚Äòrinishidagi o‚Äòzgaruvchi
    return sum(args)

# Funksiyaga 5 ta son uzatdik
print(total_sum(1, 2, 3, 4, 5))  # Natija: 15

üéØ Sizda savatdagi mahsulotlar narxini avtomatik hisoblaydigan funksiya bor.

# Funksiya: mahsulot narxlarini qabul qilib, umumiy summani hisoblaydi
def hisobla_savat(*narxlar):
    umumiy = sum(narxlar)
    return umumiy

# Mahsulotlar: non 8000, sut 12000, guruch 25000, choy 15000
print(hisobla_savat(8000, 12000, 25000, 15000))  # Natija: 60000

‚úÖ **KWARGS

üìå kwargs ‚Äî bu funksiyaga kalit-qiymat juftliklari ko‚Äòrinishida istalgancha argument yuborish imkonini beradi. U lug‚Äòat (dictionary) sifatida qabul qilinadi: key: value

# Funksiya: foydalanuvchi profilidagi har bir kalit-qiymatni chiqaradi
def print_profile(**kwargs):
    # kwargs ‚Äî dictionary, .items() bilan kalit-qiymatlarni ajratamiz
    for key, value in kwargs.items():
        print(f"{key}: {value}")

# Funksiyani chaqiramiz, kalit-qiymat juftliklari bilan
print_profile(name="Bob", age=30, profession="Engineer")

üéØ Sizda foydalanuvchini ma‚Äôlumotlar bazasiga yozish funksiyasi bor. Har bir foydalanuvchi har xil ma‚Äôlumot yuborishi mumkin: faqat ism, yoki ism + yoshi, yoki hammasi.

def save_user(**info):
    print("Yangi foydalanuvchi ma'lumotlari:")
    for key, value in info.items():
        print(f"{key}: {value}")
    print("‚úÖ Saqlandi\n")

# Turli foydalanuvchilar turlicha ma'lumot yuboradi

save_user(name="Laylo", phone="998901234567", email="laylo@mail.com")

‚úÖ LAMBDA FUNKSIYALAR

üìå lambda ‚Äî bu bir qatorli va tez yoziladigan funksiyalardir. Odatdagi def funksiyalarga o‚Äòxshaydi, lekin juda sodda va qisqa ko‚Äòrinishda yoziladi. Lambda funksiyalar odatda bir martalik, oddiy amallar uchun ishlatiladi (masalan: hisoblash, filtrlash, saralash).

# 1. Sonni kvadratga ko‚Äòtarish (x ** 2)
square = lambda x: x ** 2
print(square(6))  # 36

# 2. Ikkita sonni qo‚Äòshish (x + y)
add = lambda x, y: x + y
print(add(3, 4))  # 7

üéØ Sizda bir nechta foydalanuvchi bor, ularni yosh bo‚Äòyicha tartiblash kerak. Bunda lambda funksiyadan foydalanish mumkin.

# Foydalanuvchilar (lug‚Äòatlar ro‚Äòyxati)

    {"name": "Ali", "age": 25},
    {"name": "Laylo", "age": 20},
    {"name": "Javohir", "age": 30},

# Yosh bo‚Äòyicha tartiblash (lambda yordamida)
sorted_users = sorted(users, key=lambda user: user["age"])

# Natijani chiqaramiz
for user in sorted_users:

‚úÖ NESTED FUNCTIONS

üìå Nested function ‚Äî bu bir funksiyaning ichida boshqa funksiya yaratish demakdir. Ichki funksiya (inner) faqat tashqi funksiya (outer) ichida mavjud bo‚Äòladi va tashqaridan chaqirib bo‚Äòlmaydi. Bu usul kodni modullashtirish, xavfsizroq saqlash va faqat kerakli joyda ishlatish uchun qulay.

def outer(x):  # Tashqi funksiya
    def inner(y):  # Ichki funksiya
        return y + 2  # Ichki funksiya 2 ni qo‚Äòshadi
    return inner(x) * 2  # Inner natijasi 2 ga ko‚Äòpaytiriladi

result = outer(5)  # (5 + 2) * 2 = 14
print(result)  # 14

üéØ Siz foydalanuvchiga xush kelibsiz xabarini turli tillarda chiqarishni xohlaysiz.

def greeting(language):

# O‚Äòzbekcha salomlashuv funksiyasi

‚úÖ RECURSIVE FUNCTIONS

üìå Rekursiv funksiya ‚Äî bu o‚Äòzini o‚Äòzi ichida chaqiradigan funksiya. Bu usul, odatda takroriy (recursive) muammolarni hal qilish uchun ishlatiladi ‚Äî masalan: faktorial, Fibonachchi, fayl strukturalari, daraxt ko‚Äòrinishidagi ma‚Äôlumotlar va hokazo.

def factorial(n):
    # Bazaviy holat: n 0 bo‚Äòlsa, 1 ni qaytar
    if n == 0:
        return 1
    # Rekursiv chaqirish: n * factorial(n - 1)
    return n * factorial(n - 1)

print(factorial(5))  # 120

üéØ Kompyuterdagi papkalar ichida yana papkalar bo‚Äòlishi mumkin. Har bir darajadagi fayllarni hisoblash uchun rekursiya ishlatiladi (o‚Äòrnatilgan os moduli orqali).

import os

def count_files(folder_path):
    total = 0
    for item in os.listdir(folder_path):
        full_path = os.path.join(folder_path, item)
        if os.path.isdir(full_path):
            total += count_files(full_path)  # Ichki papkani tekshirish
        else:
            total += 1  # Fayl topildi
    return total

# Misol uchun: "/home/umid/hujjatlar"
print(count_files("/home/umid/hujjatlar"))

üìå Type annotation ‚Äî bu o‚Äòzgaruvchilar, parametrlar va return (natija) uchun ma‚Äôlumot turini ko‚Äòrsatish usulidir. Bu yordamida kod ancha tushunarli bo‚Äòladi, xatolarni aniqlash osonlashadi, IDE (VS Code, PyCharm) avtomatik tekshirish qiladi.

def add_numbers(a: int, b: int) -> int:

‚úÖ DOCSTRING

üìå Docstring (documentation string) ‚Äî bu funksiya, klass yoki modul nima qilishini tavsiflab beruvchi matn. U funksiyaning birinchi qatorida uchta qo‚Äòshtirnoq (""") bilan yoziladi. Maqsad: Funksiya qanday ishlaydi, qanday parametrlar oladi, va nima qaytaradi ‚Äî bularni tushuntirish.

def multiply(a: int, b: int) -> int:
    """
    Ikki sonni ko'paytiradi va natijani qaytaradi.
    :param a: birinchi son
    :param b: ikkinchi son
    :return: natija (int)
    """
    return a * b

print(multiply.__doc__)

‚úÖ HIGHER-ORDER FUNKSIYALAR

üìå Higher-order function ‚Äî bu boshqa funksiyani argument sifatida qabul qiladigan yoki funksiya sifatida natijada qaytaradigan funksiyadir. Pythonda funksiyalar ham "obyekt" bo‚Äòlgani uchun, ularni o‚Äòzgaruvchiga berish, funksiyaga uzatish, yoki qaytarish mumkin.

# apply_twice funksiyasi boshqa funksiya (func) va qiymat (value) oladi
def apply_twice(func, value):
    # func(value) chaqiriladi ‚Üí natijasi yana func ga beriladi
    return func(func(value))

# increment funksiyasi bitta sonni 1 ga oshiradi
def increment(x):
    return x + 1

# apply_twice funksiyasiga increment funksiyasi va 5 soni uzatilmoqda
result = apply_twice(increment, 5)
print(result)  # 7 chiqadi, chunki: increment(increment(5)) ‚Üí increment(6) ‚Üí 7

‚úÖ DECORATORS

üìå Dekorator ‚Äî bu boshqa funksiyani o‚Äòrab, unga qo‚Äòshimcha imkoniyatlar qo‚Äòshadigan higher-order funksiyadir. Asosiy funksiyani o‚Äòzgartirmasdan unga funksional qo‚Äòshish uchun ishlatiladi.

def uppercase_decorator(func):
    # Ichki funksiyani yaratamiz
    def wrapper():
        result = func()          # Asosiy funksiyani chaqiramiz
        return result.upper()    # Natijani katta harflarga o‚Äòzgartiramiz
    return wrapper               # wrapper funksiyasini qaytaramiz

@uppercase_decorator
def greet():
    return "hello"

print(greet())  # "HELLO"

üéØ APIda foydalanuvchini tekshirish (auth) uchun dekorator

# login_required ‚Äî bu dekorator funksiyasi bo‚Äòlib, boshqa funksiyani argument sifatida qabul qiladi
def login_required(func):

    # Ichki wrapper funksiyasi ‚Äî bu o‚Äòralgan yangi funksiya
    def wrapper():
        # Qo‚Äòshimcha amal: foydalanuvchining login holatini tekshirish imitatsiyasi
        print("üîê Foydalanuvchi login bo‚Äòlganligini tekshiryapmiz...")

        # Asl funksiyani chaqiramiz (masalan: view_dashboard)
        return func()

    # wrapper funksiyasini qaytaramiz ‚Äî ya‚Äôni dekoratsiya qilingan yangi funksiya
    return wrapper


# Bu yerda view_dashboard funksiyasi login_required dekoratori bilan "bezanmoqda"
# Ya'ni, view_dashboard = login_required(view_dashboard)
@login_required
def view_dashboard():
    # Asl funksiyadagi ish: dashboard ochilishini bildiradi
    print("üìä Dashboard ochildi.")
