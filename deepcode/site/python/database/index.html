‚úÖ MA'LUMOTLAR BAZASI VA PYTHON HAQIDA TUSHUNCHA

‚úÖ SQLite AFZALLIKLARI:

  • ‚úÖ Kichik va tez ishlaydi

‚úÖ DATABASE BILAN ISHLASH BOSQICHLARI

üìå SQLite bilan ishlash uchun 5 asosiy bosqich mavjud:

  • Jadval yaratish ‚Äì Ma‚Äôlumotlarni saqlash uchun jadval hosil qilish.
  • Ma‚Äôlumot qo‚Äòshish ‚Äì Bazaga yangi malumotlar kiritish.
  • Ma‚Äôlumotlarni o‚Äòqish ‚Äì Jadvaldagi ma‚Äôlumotlarni olish.
  • Ma‚Äôlumotlarni yangilash va o‚Äòchirish ‚Äì Malumotlarni o‚Äòzgartirish yoki o‚Äòchirish.

‚úÖ MA'LUMOTLARNI O'QISH

‚úÖ XATOLIKLARNI USHLASH

    # Bazaga ulanamiz
    conn = sqlite3.connect("students.db")
    # Cursor obyekti yaratamiz
    cur = conn.cursor()

    # students jadvalidan barcha ustunlar va qatorlarni olamiz

# Agar sqlite3 bilan bog‚Äòliq xatolik yuz bersa, uni ushlaymiz va xabar chiqaramiz
