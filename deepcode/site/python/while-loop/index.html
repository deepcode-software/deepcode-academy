üéØ Manfiy va nol sonlarni tashlab ketish (continue bilan)

# Indeksni 0 dan boshlaymiz

# Tsikl: indeks ro'yxat uzunligidan kichik bo‚Äòlsa davom etadi

    # Agar hozirgi son 0 yoki manfiy bo‚Äòlsa

        # Indeksni 1 ga oshiramiz, keyingi songa o‚Äòtamiz
        i += 1
        # Ushbu davrani tashlab, tsikl boshiga qaytamiz

    # Agar son musbat bo‚Äòlsa, uni ekranga chiqaramiz
    print(numbers[i])

    # Indeksni 1 ga oshirib, keyingi elementga o‚Äòtamiz
    i += 1
