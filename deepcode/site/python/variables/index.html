‚úÖ O'ZGARUVCHI NIMA?

üìå Python dasturlash tilida o‚Äòzgaruvchi (variable) ‚Äî bu kompyuterda ma‚Äôlumotni vaqtincha saqlab turadigan nomlangan joy. Bu joyga biz har qanday ma‚Äôlumotni (masalan, matn, son) joylashtirishimiz mumkin. Keyin bu nom orqali o‚Äòsha ma‚Äôlumotdan foydalanish, o‚Äòzgartirish yoki hisoblash osonlashadi. O‚Äòzgaruvchilar bizga dastur ichida ma‚Äôlumot bilan ishlashni tartibli va qulay qiladi. Har safar ma‚Äôlumotni qayta yozmasdan, nomini chaqirib ishlatishimiz mumkin.

üìå Quyidagi misolda 4 ta o'zgaruvchi yaratdik (x, y, name va is_student) va ularga har xil ma'lumot yukladik.

# Butun sonni (integer) o'zgaruvchiga saqlaymiz
x = 5

# Haqiqiy sonni (float) o'zgaruvchiga saqlaymiz
y = 3.14

# Matn (string) qiymatni o'zgaruvchiga saqlaymiz
name = "Alice"

# Mantiqiy (boolean) qiymatni o'zgaruvchiga saqlaymiz
is_student = True

# x o'zgaruvchisining qiymatini chiqaramiz
print(x)

# y o'zgaruvchisining qiymatini chiqaramiz
print(y)

# name o'zgaruvchisining qiymatini chiqaramiz
print(name)

# is_student o'zgaruvchisining qiymatini chiqaramiz
print(is_student)

üìå variable diyilishini sababi uning qiymati istalgan payt o'zgarishi mumkin:

# 'name' o'zgaruvchisiga dastlab 'Alisher' matnini beramiz
name = 'Alisher'

# name o'zgaruvchisining hozirgi qiymatini chiqaramiz (Alisher)
print(name)

# name o'zgaruvchisining qiymatini o'zgartiramiz, endi u 'Muhammad' bo'ladi
name = "Muhammad"

# name o'zgaruvchisining yangi qiymatini chiqaramiz (Muhammad)
print(name)

‚úÖ O'ZGARUVCHILARNI NOMLASH

‚ùó O'zgaruvchilarga nom berishda quyidagi qoidalarga amal qiling:

‚ùå O'zgaruvchi nomi harf yoki pastki chiziq (_) bilan boshlanishi kerak

‚ùå Noto‚Äòg‚Äòri:

# ‚ùå Son bilan boshlanmaydi

‚ùå O'zgaruvchi nomi raqam bilan boshlanishi mumkin emas

üìå Raqam faqat nomning oxirida yoki o‚Äòrtasida ishlatilishi mumkin.

‚úÖ To‚Äòg‚Äòri:

# O'zgaruvchi nomi harf bilan boshlangan va oxirida raqam ishlatilgan ‚Äî to'g'ri

‚ùå Noto‚Äòg‚Äòri:

‚ùå O'zgaruvchi nomida faqatgina ingliz tili alifbosi harflari (A-z), raqamlar (0-9) va pastki chiziq (_) qatnashishi mumkin

üìå Maxsus belgilar (@, !, #, - va boshqalar) o‚Äòzgaruvchi nomida ishlatilmaydi.

‚úÖ To‚Äòg‚Äòri:

# Harf va raqam ishlatilgan ‚Äî to‚Äòg‚Äòri

# Pastki chiziq bilan boshlangan nom ‚Äî bu ham to‚Äòg‚Äòri

‚ùå Noto‚Äòg‚Äòri:

# O'zgaruvchi nomida '@' belgisi ishlatilgan ‚Äî bu noto‚Äòg‚Äòri
# ‚ùå Maxsus belgilar (masalan: @) Python'da o'zgaruvchi nomida bo'lmasligi kerak

# O'zgaruvchi nomida '!' belgisi ishlatilgan ‚Äî bu ham noto‚Äòg‚Äòri
# ‚ùå Maxsus belgilar (masalan: !) ruxsat etilmaydi

‚ùå O'zgaruvchi nomida bo'shliq (–ø—Ä–æ–±–µ–ª) bo'lishi mumkin emas

‚úÖ To‚Äòg‚Äòri:

‚ùå Noto‚Äòg‚Äòri:

# O'zgaruvchi nomida bo‚Äòshliq (space) ishlatilgan ‚Äî bu noto‚Äòg‚Äòri
# ‚ùå Python'da o'zgaruvchi nomi bo‚Äòshliq bilan yozilmaydi

‚ùå O'zgaruvchi nomida katta-kichik harflar turlicha talqin qilinadi (ism, ISM, va Ism uchta turli o'zgaruvchi)

üìå Pythonda ism, ISM va Ism bu uchta alohida o‚Äòzgaruvchi hisoblanadi.

# kichik harflardan tashkil topgan o'zgaruvchi

# hamma harflari katta bo'lgan o'zgaruvchi ‚Äî bu boshqa o'zgaruvchi

# bosh harfi katta, qolgan kichik bo'lgan o'zgaruvchi ‚Äî yana boshqa o'zgaruvchi

# 'ism' o'zgaruvchisining qiymatini chiqaramiz

# 'ISM' o'zgaruvchisining qiymatini chiqaramiz
print(ISM)  # Vali

# 'Ism' o'zgaruvchisining qiymatini chiqaramiz
print(Ism)  # Sami

‚úÖ QO'SHIMCHA QOIDALAR

üìå Python kodini o‚Äòqishda va tushunishda qulaylik uchun o‚Äòzgaruvchilarni kichik harflar bilan yozish odatiy hisoblanadi.

‚ùáÔ∏è Shuningdek o'zgaruvchilarga Pythonda ishlatiladigan funksiyalar va maxsus kalit so'zlarning(keywords) nomini bermang. Kalit so'zlar ro'yhatini ko'rish uchun python faylga uyidagi kodni yozamiz:
