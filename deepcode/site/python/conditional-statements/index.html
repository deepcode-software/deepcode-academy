TARMOQLANISH

üìå Shu vaqtgacha yozgan dasturlarimizga e'tibor bersangiz, dasturimiz yuqoridan pastga qarab qatorma-qator bajarilib keldi. Bu chiziqli dastur deyiladi. Ammo ba'zida shartga qarab kodning bir qismidan boshqa qismiga o'tish zarur bo'ladi. Bunday holat "tarmoqlanish" deb ataladi.

üìå Python dasturlash tilida shart operatorlari (conditional statements) dasturda turli vaziyatlarga qarab turli amallarni bajarishga imkon beradi.

‚úÖ IF

üìå Python dasturlash tilida if operatori shartni tekshirish uchun ishlatiladi. Ya'ni, agar biror shart True bo‚Äòlsa, unga tegishli kodlar bajariladi. Agar shart False bo‚Äòlsa, kod bajarilmaydi.

# Bankdagi hisobdagi pul miqdori
account_balance = 1500  

# Agar balans 1000 dan katta bo‚Äòlsa, foydalanuvchi pul yechishi mumkin
if account_balance > 1000:
    print("You can withdraw money")  # Foydalanuvchiga ruxsat beriladi

# Foydalanuvchining savatidagi umumiy mahsulot narxi
total_price = 250  

# Agar narx 200 dan katta bo‚Äòlsa, bepul yetkazib berish taklif qilinadi
if total_price > 200:
    print("You qualify for free shipping")  # Bepul yetkazib berish haqida xabar

# Foydalanuvchi kiritgan parol
password = "mysecurepass"  

# Agar parol uzunligi 8 belgidan ko‚Äòp bo‚Äòlsa, kuchli parol deb baholanadi
if len(password) > 8:
    print("Your password is strong")  # Kuchli parol haqida xabar

‚úÖ ELSE

üìå Python dasturlash tilida else operatori if dan keyin yoziladi. Agar if dagi False bo;lsa, else ichidagi kodlar bajariladi.

# Foydalanuvchi shakarmi tanladi (ha yoki yo'q)
wants_sugar = True  

# Agar foydalanuvchi shakar istasa
if wants_sugar:
    print("Adding sugar to your coffee")  # Qahvaga shakar qo‚Äòshiladi
else:
    print("Preparing your coffee without sugar")  # Shakarsiz qahva tayyorlanadi

# Telefon quvvati foizda
battery_percentage = 15  

# Agar quvvat 20 dan kam bo‚Äòlsa, ogohlantirish chiqariladi
if battery_percentage < 20:
    print("Battery is low. Please charge your phone.")  # Quvvat kamligi haqida ogohlantirish
else:
    print("Battery level is sufficient.")  # Quvvat yetarli

# Talabaning olgan bahosi
exam_score = 72  

# Agar baho 60 yoki undan yuqori bo‚Äòlsa, imtihondan o‚Äòtgan hisoblanadi
if exam_score >= 60:
    print("You passed the exam!")  # O‚Äòtdi degan xabar
else:
    print("You failed the exam.")  # Yiqildi degan xabar

‚úÖ ELIF

elif ‚Äî bu else if degan so‚Äòzning qisqargan ko‚Äòrinishi. U bir nechta shartlarni ketma-ket tekshirish uchun ishlatiladi. Agar if sharti False bo'lsa, elif sharti tekshiriladi.

üéØ Smart uydagi HVAC tizimi xonadagi haroratga qarab qaror qabul qiladi.

# Current room temperature in Celsius
room_temperature = 28  # Xona harorati, Celsiy bo‚Äòyicha

# Agar xona juda sovuq bo‚Äòlsa (18¬∞C dan past)
if room_temperature < 18:
    print("Heating system should be turned ON")  # Isitish tizimi yoqiladi

# Agar xona juda issiq bo‚Äòlsa (26¬∞C dan yuqori)
elif room_temperature > 26:
    print("Cooling system should be turned ON")  # Sovutish tizimi yoqiladi

# Aks holda, harorat me'yorda
else:
    print("Temperature is optimal, no action needed")  # Hech qanday tizim yoqilmaydi

üéØ Onlayn do‚Äòkonda har bir buyurtmaning holatini aniqlash uchun ishlatiladi.

# Order status (current stage of delivery process)

# Agar buyurtma qabul qilingan bo‚Äòlsa
if order_status == "received":
    print("Your order has been received and will be prepared soon.")  # Buyurtma qabul qilindi

# Agar tayyorlanayotgan bo‚Äòlsa

    print("Your order is currently being prepared.")  # Tayyorlanmoqda

# Agar yuborilgan bo‚Äòlsa
elif order_status == "shipped":
    print("Your order has been shipped.")  # Yo‚Äòlga chiqdi

# Agar yetkazilgan bo‚Äòlsa
elif order_status == "delivered":
    print("Your order has been delivered.")  # Yetkazildi

# Nomalum holatlar uchun
else:
    print("Unknown order status.")  # Noma'lum status

‚úÖ BIR NECHTA SHARTLARNI BIRGA ISHLATISH

üéØ Login tizimi ‚Äì foydalanuvchini tekshirish. Foydalanuvchi login va parolni to‚Äòg‚Äòri kiritganmi, tekshiramiz.

# Foydalanuvchi kiritgan login va parol

entered_password = "python123"

# To‚Äòg‚Äòri login va parol kiritilgan bo‚Äòlsa
if entered_username == "umid" and entered_password == "python123":
    print("Welcome, Umid!")  # Xush kelibsiz!

# Login to‚Äòg‚Äòri, lekin parol noto‚Äòg‚Äòri bo‚Äòlsa
elif entered_username == "umid" and entered_password != "python123":
    print("Incorrect password")  # Parol xato

# Parol to‚Äòg‚Äòri, lekin login noto‚Äòg‚Äòri bo‚Äòlsa
elif entered_username != "umid" and entered_password == "python123":
    print("Incorrect username")  # Login xato

# Ikkalasi ham noto‚Äòg‚Äòri bo‚Äòlsa
else:
    print("Incorrect username and password")  # Ikkalasi ham xato

üéØ Ob-havo ilovasi ‚Äì kiyim tavsiyasi. Foydalanuvchiga havo haroratiga qarab tavsiya beriladi.

# Ob-havo holati: havo harorati va yomg‚Äòir yog‚Äòyaptimi yo‚Äòqmi

is_raining = False  # Yomg‚Äòir yog‚Äòayotgan bo‚Äòlsa True, aks holda False

# Juda sovuq bo‚Äòlsa (0¬∞C yoki undan past)

    print("It is very cold. Wear warm clothes.")  # Qalin kiyim tavsiya qilinadi

# Salqin bo‚Äòlsa (1¬∞C dan 15¬∞C gacha)
elif temperature <= 15:
    print("It's cool outside. A light jacket is recommended.")  # Yengil kurtka kiying

# Juda issiq yoki yomg‚Äòirli bo‚Äòlsa
elif temperature >= 30 or is_raining:
    print("Hot or rainy weather ‚Äì wear light clothes and carry an umbrella.")  # Soyabon va yengil kiyim

# Ob-havo normal bo‚Äòlsa
else:
    print("The weather is normal. Regular clothes are fine.")  # Oddiy kiyim yetarli

üéØ Internet tezligiga qarab videoning sifat darajasi belgilanadi.

‚úÖ NESTED CONDITIONS

üìå Shart operatorlarini bir-birining ichiga joylashtirish orqali murakkabroq mantiqiy holatlar yaratish mumkin.

üéØ Foydalanuvchi login kiritadi, agar mavjud bo‚Äòlsa, yoshi tekshiriladi. Faqat voyaga yetganlarga ruxsat.

# Foydalanuvchi nomi (sistemaga kirishga urinayotgan)
username = "umid"

# Foydalanuvchilar ro'yxati (username: age)
users = {
    "umid": 21,
    "ali": 17
}

# Avvalo foydalanuvchi ro'yxatda bormi ‚Äì tekshiramiz
if username in users:
    age = users[username]  # Username bo‚Äòyicha yoshni olamiz

    # Agar foydalanuvchi 18 yoki undan katta bo‚Äòlsa ‚Äì ruxsat beriladi
    if age >= 18:
        print(f"{username.title()} is allowed to access the system.")  # Ruxsat berildi

    # Agar foydalanuvchi voyaga yetmagan bo‚Äòlsa ‚Äì ruxsat berilmaydi
    else:
        print(f"{username.title()} is underage. Access denied.")  # Ruxsat yo‚Äòq
else:
    print("User not found in the system.")  # Bunday foydalanuvchi yo‚Äòq

üéØ Foydalanuvchi balansiga qarab faqat ruxsat etilgan tranzaksiyani bajarish mumkin.

# User's current balance
balance = 500_000  # 500 ming so‚Äòm

# Transaction type: "withdraw" (yechish) yoki "deposit" (kiritish)

# Transaction amount (so‚Äòmda)
amount = 300_000

# Agar foydalanuvchi pul yechmoqchi bo‚Äòlsa
if transaction_type == "withdraw":
    # Mablag' yetarlimi? Yechishga ruxsat beriladi
    if balance >= amount:
        print(f"{amount} so'm has been successfully withdrawn.")  # Pul yechildi
    else:
        print("Insufficient balance.")  # Pul yetarli emas

# Agar foydalanuvchi balansga pul kiritmoqchi bo‚Äòlsa
elif transaction_type == "deposit":
    print(f"{amount} so'm has been added to your balance.")  # Pul kiritildi

# Aks holda, tranzaksiya turi noto‚Äòg‚Äòri
else:
    print("Invalid transaction type.")  # Noma'lum amal

üéØ Agar foydalanuvchi o‚Äòqituvchi bo‚Äòlsa, yangi kurs yarata oladi. Talaba bo‚Äòlsa, faqat yozilishi mumkin.

# Foydalanuvchining ma'lumotlari (ismi, roli, holati)

    "name": "Umid",
    "role": "student",     # "student" yoki "teacher"
    "active": True         # Foydalanuvchi faolligi
}

# Agar foydalanuvchi faollashtirilgan bo‚Äòlsa

    # Agar foydalanuvchi o‚Äòqituvchi bo‚Äòlsa
    if user["role"] == "teacher":
        print("You can create a new course.")  # Yangi kurs yaratish huquqi bor

    # Agar foydalanuvchi talaba bo‚Äòlsa
    elif user["role"] == "student":
        print("You can enroll in a course.")  # Kursga yozilish huquqi bor

    # Boshqa noma'lum rollar uchun
    else:
        print("Your role is not recognized.")  # Rol noto‚Äòg‚Äòri yoki mavjud emas

# Agar foydalanuvchi faollashtirilmagan bo‚Äòlsa
else:
    print("User is not active.")  # Foydalanuvchi faol emas

‚úÖ MULTIPLE LEVELS OF NESTED CONDITIONAL STATEMENTS

üéØ Ma'lumotlar bazasida foydalanuvchi ro‚Äòyxatdan o‚Äòtganmi, email tasdiqlanganmi va yoshi to‚Äòliqmi ‚Äî shu tartibda tekshiriladi:

# Foydalanuvchining ro‚Äòyxatdan o‚Äòtganligi, email holati va yoshi

    "username": "umid",
    "registered": True,         # Foydalanuvchi ro'yxatdan o'tganmi?
    "email_verified": True,     # Email tasdiqlanganmi?
    "age": 19                   # Foydalanuvchi yoshi
}

# 1. Avval foydalanuvchi ro‚Äòyxatdan o‚Äòtganligini tekshiramiz
if user["registered"]:

    # 2. Email tasdiqlanganligini tekshiramiz
    if user["email_verified"]:

        # 3. Yoshi 18 yoki undan katta bo‚Äòlsa ‚Äì to‚Äòliq ruxsat beriladi
        if user["age"] >= 18:
            print("Full access granted to the system.")  # To‚Äòliq ruxsat

        else:
            print("Access denied. You must be at least 18 years old.")  # Yoshi yetmaydi

    else:
        print("Email not verified. Please verify your email first.")  # Email tasdiqlanmagan

else:
    print("You are not registered. Please sign up first.")  # Ro‚Äòyxatdan o‚Äòtmagan

üéØ Avtomobil ijarasi faqat quyidagi shartlar bajarilganda mumkin:

  • Foydalanuvchi ro‚Äòyxatdan o‚Äòtgan
  • Haydovchilik guvohnomasi mavjud
  • Yoshi 21 dan katta
# User data: registration, license availability, and age
user = {
    "registered": True,       # Foydalanuvchi ro‚Äòyxatdan o‚Äòtganmi?
    "has_license": True,      # Haydovchilik guvohnomasi bormi?
    "age": 25                 # Foydalanuvchining yoshi
}

# 1. Foydalanuvchi ro‚Äòyxatdan o‚Äòtgan bo‚Äòlishi kerak
if user["registered"]:

    # 2. Haydovchilik guvohnomasi mavjud bo‚Äòlishi kerak
    if user["has_license"]:

        # 3. Yoshi 21 dan katta yoki teng bo‚Äòlishi kerak
        if user["age"] >= 21:
            print("Car rental approved.")  # Ijaraga ruxsat berildi
        else:
            print("You must be at least 21 years old to rent a car.")  # Yoshi yetmaydi

    else:
        print("You must have a valid driver‚Äôs license.")  # Guvohnoma yo‚Äòq

else:
    print("Please register before renting a car.")  # Ro‚Äòyxatdan o‚Äòtmagan

‚úÖ AMALIYOT

‚úÖ 1-topshiriq:

Foydalanuvchidan yoshini so‚Äòrang. Agar 18 dan kichik bo‚Äòlsa "Voyaga yetmagan", 18 yoki undan katta bo‚Äòlsa "Voyaga yetgan" deb chiqaring.

‚úÖ 2-topshiriq:

Quyidagi o‚Äòzgaruvchi berilgan: 

score = 87

Agar score 90 va undan yuqori bo‚Äòlsa "A", 80-89 oralig‚Äòida bo‚Äòlsa "B", 70-79 oralig‚Äòida bo‚Äòlsa "C", boshqacha bo‚Äòlsa "D" baho chiqaring.

‚úÖ 3-topshiriq:

Quyidagi o‚Äòzgaruvchi berilgan: 

is_raining = True

Agar is_raining True bo‚Äòlsa "Soyabon oling", aks holda "Yomg‚Äòir yo‚Äòq" deb chiqaring.

‚úÖ 4-topshiriq:

Quyidagi ro‚Äòyxat berilgan: 

fruits = ["olma", "banan", "gilos"]

Agar ro‚Äòyxatda "banan" bo‚Äòlsa "Banan bor", bo‚Äòlmasa "Banan yo‚Äòq" deb chiqaring.

‚úÖ 5-topshiriq:

Ro‚Äòyxatda kamida 5 ta element bo‚Äòlsa "Ro‚Äòyxat to‚Äòla", aks holda "Ro‚Äòyxat qisqa" deb chiqaring: 

nums = [1, 4, 7, 10]

‚úÖ 6-topshiriq:

Tuple berilgan: 

location = (41.2, 69.1)

Agar location bo‚Äòsh bo‚Äòlmasa "Joylashuv aniqlangan", bo‚Äòsh bo‚Äòlsa "Joylashuv yo‚Äòq" deb chiqaring.

‚úÖ 7-topshiriq:

Set berilgan: 

active_ids = {1, 2, 3}

Agar 2-ID setda mavjud bo‚Äòlsa "2-ID mavjud", bo‚Äòlmasa "2-ID yo‚Äòq" deb chiqaring.

‚úÖ 8-topshiriq:

Quyidagi dictionary berilgan: 

profile = {"name": "Alim", "age": 15}

Agar age 18 yoki undan katta bo‚Äòlsa "Voyaga yetgan", aks holda "Voyaga yetmagan" deb chiqaring.

‚úÖ 9-topshiriq:

Quyidagi o‚Äòzgaruvchilar berilgan: 

is_logged_in = True
is_admin = False

Agar foydalanuvchi tizimga kirgan va admin bo‚Äòlsa "Admin panelga xush kelibsiz", faqat tizimga kirgan bo‚Äòlsa "Foydalanuvchi paneli", har ikkisi ham False bo‚Äòlsa "Kirish talab qilinadi".

‚úÖ 10-topshiriq:
