üåê DJANGO REST FRAMEWORK ASOSLARI

üß© 1-DARS: DJANGO LOYIHASI VA DRF SOZLASH

Bu darsda Django va Django REST Framework (DRF) yordamida loyiha yaratish va oddiy API sozlashni bosqichma-bosqich o'rganamiz. Har bir qadam tushunarli va aniq bo'ladi.

‚úÖ 1. MUHITNI TAYYORLASH

‚ùáÔ∏è Python o'rnatish:

üìå Python 3.8 yoki undan yuqori versiyani o'rnating (https://www.python.org/downloads/).

‚ùáÔ∏è Virtual muhit yaratish:

üìå Virtual muhit loyiha uchun alohida muhit yaratadi, bu boshqa loyihalarga ta'sir qilmaslikni ta'minlaydi.

python -m venv env
source env/bin/activate  # Windows uchun: env\Scripts\activate

‚ùáÔ∏è Django va DRF o'rnatish:

üìå Bu Django va DRF kutubxonalarini o'rnatadi.

pip install django djangorestframework

‚úÖ 2. DJANGO LOYIHASINI YARATISH

‚ùáÔ∏è Loyiha yaratish:

üìå Quyidagi buyruq myproject nomli yangi loyiha jildini yaratadi.

django-admin startproject myproject .

‚ùáÔ∏è Loyha tuzilishi:

myproject/

‚ùáÔ∏è Serverni ishga tushirish:

‚úÖ 3. DJANGO ILOVASINI YARATISH

üìå Django loyihasida ilovalar (apps) alohida modullar sifatida ishlaydi.

‚ùáÔ∏è Ilova yaratish:

python manage.py startapp myapp

üìå Bu myapp nomli ilova jildini yaratadi.

‚ùáÔ∏è Ilovani loyihaga qo'shish:

üìå myproject/settings.py faylini oching va INSTALLED_APPS ro'yxatiga quyidagini qo'shing:

INSTALLED_APPS = [
    ...
    'myapp',
    'rest_framework',  # DRF ni qo'shamiz
]

‚úÖ 4. MA'LUMOTLAR BAZASINI SOZLASH

üìå Django standart ravishda SQLite bilan ishlaydi, bu oddiy loyihalar uchun yetarli.

‚ùáÔ∏è Model yaratish:

üìå myapp/models.py faylida oddiy model yaratamiz:

from django.db import models

class Item(models.Model):
    name = models.CharField(max_length=100)
    description = models.TextField()
    created_at = models.DateTimeField(auto_now_add=True)

    def __str__(self):
        return self.name

‚ùáÔ∏è Migratsiyalarni yaratish va qo'llash:

üìå Modelni ma'lumotlar bazasida jadvallar sifatida yaratish uchun:

python manage.py makemigrations
python manage.py migrate

‚úÖ 5. DRF BILAN API YARATISH

üìå DRF yordamida oddiy API yaratamiz.

‚ùáÔ∏è Serializer yaratish:

üìå myapp/serializers.py faylini yarating va quyidagi kodni qo'shing:

from rest_framework import serializers
from .models import Item

class ItemSerializer(serializers.ModelSerializer):
    class Meta:
        model = Item
        fields = ['id', 'name', 'description', 'created_at']

‚ùáÔ∏è View yaratish:

üìå myapp/views.py faylida API view yaratamiz:

from rest_framework import viewsets
from .models import Item
from .serializers import ItemSerializer

class ItemViewSet(viewsets.ModelViewSet):
    queryset = Item.objects.all()
    serializer_class = ItemSerializer

‚ùáÔ∏è URL sozlash:

üìå myproject/urls.py faylini yangilang:

from django.urls import path, include
from rest_framework.routers import DefaultRouter
from myapp.views import ItemViewSet

router = DefaultRouter()
router.register(r'items', ItemViewSet)

urlpatterns = [
    path('', include(router.urls)),
]

‚úÖ 6. API NI SINAB KO'RISH

üìå Serverni ishga tushiring:

python manage.py runserver

üìå Brauzerda http://127.0.0.1:8000/items/ manziliga o'ting. DRF interfeysi orqali ma'lumotlarni ko'rishingiz, qo'shishingiz va o'zgartirishingiz mumkin.

‚úÖ 7. MA'LUMOT QO'SHISH (IXTIYORIY)

üìå Django admin paneli orqali ma'lumot qo'shish uchun:

‚ùáÔ∏è Admin foydalanuvchisini yaratish:

python manage.py createsuperuser

‚ùáÔ∏è Modelni admin panelida ro'yxatdan o'tkazish:

üìå myapp/admin.py faylida modelni ro'yxatdan o'tkazing:

from django.contrib import admin
from .models import Item

admin.site.register(Item)

üìå http://127.0.0.1:8000/admin/ manzilida admin paneliga kiring va ma'lumot qo'shing.
